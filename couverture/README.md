# Rapport de couverture (Backend)

Le rapport JaCoCo est généré automatiquement lors de l'exécution des tests Maven.

## Générer le rapport

```bash
./mvnw test
```

Le rapport HTML est disponible dans : `target/site/jacoco/index.html`

Copier le dossier `target/site/jacoco/` dans ce répertoire pour l'intégrer au dépôt.
